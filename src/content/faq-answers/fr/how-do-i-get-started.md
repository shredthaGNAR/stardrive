**Lancez `npm create stardrive@latest` et suivez les indications.**

**Vous pouvez aussi cloner ou forker le [dépôt](https://github.com/peltmonger/stardrive) directement et lancer `npm install`.**

Démarrez le serveur de développement avec `npm run dev`.

À partir de là, ajustez `theme.config.ts` à votre marque, remplacez les favicons et les images sociales dans `/public`, et modifiez les collections de contenu dans `/src/content` selon vos besoins.

Si vous utilisez un assistant de code IA, dirigez-le vers le dépôt et le fichier `AGENTS.md` - Stardrive est conçu pour être repris et étendu par des LLMs dès le départ.

**Consultez la section docs pour des indications plus détaillées sur la configuration, la structure et le déploiement.**
