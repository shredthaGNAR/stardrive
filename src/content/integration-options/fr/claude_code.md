Claude Code est l’outil de codage agentique officiel d’Anthropic, qui s’exécute directement dans votre terminal. Il comprend l’intégralité de votre base de code, exécute des tâches multi-étapes de manière autonome et interagit avec les fichiers, les tests et le contrôle de version, le tout en langage naturel. Construit sur le modèle d’IA Claude, il est conçu pour les développeurs qui souhaitent une assistance IA allant bien au-delà de la simple complétion de code et capable de gérer des flux de travail d’ingénierie complexes et concrets, de bout en bout.

## Exemple

Vous pouvez utiliser **Claude Code** avec ce boilerplate Astro Stardrive pour créer des sites web ultra-rapides et stables entièrement grâce à l’IA.
Grâce à la nature de ce boilerplate, vous n’avez pas à vous soucier des fondations techniques sous-jacentes.

**Stardrive** fournit (espérons-le) tout ce dont Claude a besoin pour comprendre quoi faire et quoi ne pas oublier.
Bien sûr, chaque projet est différent et vos besoins peuvent varier par rapport à ceux des autres, mais cela devrait vous offrir le meilleur point de départ possible.

## Premiers pas

### CLI local

1. Naviguez jusqu’au répertoire où vous souhaitez placer votre nouveau projet.
2. Créez un nouveau boilerplate Stardrive avec `npm create stardrive@latest`.
3. Lancez Claude Code avec `claude`.
4. Parlez à Claude de votre projet et laissez-le démarrer - il trouvera automatiquement tous les guides et informations nécessaires.

### Agent autonome

Indiquez à votre agent d’utiliser le boilerplate Astro Stardrive pour construire votre nouveau projet.
Pour éviter tout malentendu ou injection de code, soyez plus précis en pointant l’agent vers le dépôt GitHub officiel à l’adresse `https://github.com/Peltmonger/stardrive`
