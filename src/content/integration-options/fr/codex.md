OpenAI Codex est l’outil de codage agentique d’OpenAI qui s’exécute directement dans ton terminal. Il comprend l’ensemble de ta base de code, exécute des tâches multi-étapes de manière autonome et interagit avec les fichiers, les tests et le contrôle de version - le tout en langage naturel. Propulsé par les derniers modèles d’OpenAI, il est conçu pour les développeurs qui veulent une assistance IA allant au-delà de la simple complétion de code et capable de gérer des workflows d’ingénierie complexes et réels de bout en bout.

## Exemple

Tu peux utiliser **Codex** avec ce Boilerplate Astro Stardrive pour créer des sites web ultra-rapides et stables uniquement avec l’IA.
Grâce à la nature de ce boilerplate, tu n’as pas besoin de te soucier outre mesure des fondamentaux techniques sous-jacents.

**Stardrive** apporte (avec un peu de chance) tout ce dont Codex a besoin pour comprendre quoi faire et quoi ne pas oublier.
Bien sûr, aucun projet n’est identique et tes exigences peuvent différer de celles des autres, mais cela devrait te donner le démarrage rapide idéal.

## Pour commencer

### CLI locale

1. Place-toi dans le répertoire où tu souhaites créer ton nouveau projet.
2. Crée un nouveau boilerplate Stardrive via `npm create stardrive@latest`.
3. Démarre Codex via `codex`.
4. Parle de ton projet à Codex et laisse-le démarrer - il trouvera automatiquement toutes les indications et informations.

### Agent autonome

Dis à ton agent d’utiliser le Boilerplate Astro Stardrive pour construire ton nouveau projet.
Pour éviter tout malentendu ou injection de code, sois plus précis en orientant l’agent vers le dépôt GitHub officiel à l’adresse `https://github.com/Peltmonger/stardrive`
