Claude Code es la herramienta oficial de codificación agéntica de Anthropic que se ejecuta directamente en tu terminal. Comprende toda tu base de código, ejecuta tareas de múltiples pasos de forma autónoma e interactúa con archivos, pruebas y control de versiones, todo mediante lenguaje natural. Desarrollada sobre el modelo de IA Claude, está pensada para desarrolladores que buscan una asistencia de IA que vaya más allá del simple completado de código y que pueda gestionar flujos de trabajo de ingeniería complejos y reales de principio a fin.

## Ejemplo

Puedes usar **Claude Code** junto con este Astro Stardrive Boilerplate para crear sitios web ultrarrápidos y estables utilizando únicamente IA.
Gracias a la naturaleza de este boilerplate, no tienes que preocuparte demasiado por los fundamentos técnicos subyacentes.

**Stardrive** proporciona (con suerte) todo lo que Claude necesita para entender qué hacer y qué no olvidar.
Por supuesto, cada proyecto es diferente y tus requisitos pueden variar respecto a los de otros, pero esto debería darte el punto de partida ideal.

## Primeros pasos

### CLI local

1. Navega hasta el directorio donde quieres crear tu nuevo proyecto.
2. Crea un nuevo boilerplate de Stardrive con `npm create stardrive@latest`.
3. Inicia Claude Code con `claude`.
4. Cuéntale a Claude sobre tu proyecto y deja que empiece: encontrará toda la información y orientación automáticamente.

### Agente autónomo

Indica a tu agente que utilice el Astro Stardrive Boilerplate para construir tu nuevo proyecto.
Para evitar malentendidos o inyecciones de código, sé más preciso apuntando al agente al repositorio oficial de GitHub en `https://github.com/Peltmonger/stardrive`
